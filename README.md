# Country Dashboard

A **responsive, production-ready dashboard** built with **React + TypeScript + RTK Query + Bootstrap + MSW**.

---

## Features

| Feature | Status |
|-------|--------|
| Login with Formik + Yup | Complete |
| Password: 8+ chars, A, 1, ! | Complete |
| Protected Routes | Complete |
| Country List from `restcountries.com` | Complete |
| Filter: All / Asia / Europe | Complete |
| Reusable Carousel Component | Complete |
| Infinite Scroll + Load More | Complete |
| Mobile-First Design | Complete |
| MSW Mocking | Complete |
| TypeScript | Complete |

---

## Tech Stack

- **React 18** + **TypeScript**
- **Vite** (blazing fast)
- **React Router** (protected routes)
- **Redux Toolkit + RTK Query**
- **Bootstrap 5** (responsive UI)
- **Formik + Yup** (form validation)
- **MSW** (offline API mocking)
- **React Bootstrap**
